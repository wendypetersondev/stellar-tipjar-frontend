# Stellar Tip Jar Brand Guidelines

## Overview
Stellar Tip Jar is a modern Web3 tipping platform built on the Stellar network. Our brand represents trust, innovation, and community in the crypto space.

## Logo

### Primary Logo
- Full color version with gradient
- Use on light backgrounds
- Minimum size: 200px width
- Clear space: 1x logo height around all sides

### Secondary Logo
- Monochrome version for simple applications
- Use when color is not available
- Maintains readability

### Logo Icon
- Standalone icon for favicons, app icons
- Square format, scalable
- Primary blue color (#0066ff)

### Usage Rules
- Do not distort, rotate, or modify the logo
- Maintain aspect ratio
- Do not place on busy backgrounds
- Ensure sufficient contrast

## Color Palette

### Primary Colors
- Stellar Blue: #0066ff (RGB: 0, 102, 255)
- Stellar Cyan: #00d4ff (RGB: 0, 212, 255)

### Secondary Colors
- Success Green: #10b981 (RGB: 16, 185, 129)
- Warning Yellow: #f59e0b (RGB: 245, 158, 11)
- Error Red: #ef4444 (RGB: 239, 68, 68)

### Neutral Colors
- Dark Gray: #1f2937 (RGB: 31, 41, 55)
- Medium Gray: #6b7280 (RGB: 107, 114, 128)
- Light Gray: #d1d5db (RGB: 209, 213, 219)
- White: #ffffff (RGB: 255, 255, 255)

## Typography

### Primary Font: Inter
- Headings: Inter Bold (font-weight: 700)
- Body: Inter Regular (font-weight: 400)
- Sizes: 14px base, 16px body, 18px-32px headings

### Monospace Font: JetBrains Mono
- For addresses, hashes, code
- Regular weight

### Line Heights
- Body: 1.6
- Headings: 1.2

## Tone and Voice
- Professional yet approachable
- Technical accuracy
- Community-focused
- Empowering users in Web3

## Visual Style
- Clean, modern design
- Subtle shadows and gradients
- Consistent spacing (8px grid)
- High contrast for accessibility

## File Formats
- SVG for logos and icons
- PNG for raster images
- ICO for favicons
- Ensure all assets are optimized for web
